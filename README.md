# healing_blog
HTML, CSS, JavaScript를 이용해 개발한 웹 블로그 입니다.  
***  
<br/>

## ✔개요
2022년도 1학기 '웹 프로그래밍'수업에서 진행한 프로젝트이다. 나에게 힐링되는 모든 것에 대해 담은 웹 블로그이다. 메뉴에는 홈, 글보기(대학생활), 코딩이야기(코딩일지), 캘린더, 방명록으로 구성되어 있으며 수업에 맞게 프론트에만 집중해 웹 사이트가 구현되었다.
***  
<br/> 

## ✔개발
- VSCode
- HTML
- CSS
- JavaScript
***

- 메뉴에는 홈, 글보기, 코딩이야기, 캘린더 방명록이 있다.

- 메인 홈의 하단에는 간략한 나의 소개를 볼 수 있다.
